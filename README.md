## 技术点

- Redis
- RedisModule
- RedisService
- redis hash
- SessionModule
- SessionService
- cookie-parser
- nginx 负载均衡

## 区别

- JWT 主要用于无状态的身份验证和授权
- 分布式 Session 则用于在服务器端维护丰富的用户会话状态。比如在电商系统中，用户的购物车状态、浏览历史等信息
